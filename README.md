# TaskOrbit

**ADHD-Focused Productivity App**

TaskOrbit is a productivity app designed specifically for people with ADHD, featuring focus timers, task management, habit tracking, and more.

## Download

**[Download Latest Release](https://github.com/raveuk/TaskOrbit/releases/latest)**

Also available on [Google Play Store](https://play.google.com/store/apps/details?id=com.zimbabeats.taskorbit)

## Features

- Pomodoro Focus Timer with alarm notifications
- Task Management with AI-powered task breakdown
- Habit Tracking with streaks
- Daily & Weekly Planner
- Voice-to-Task (offline speech recognition)
- Virtual Pet Companion
- Breathing Exercises
- Location-based Reminders
- Premium subscription for unlimited features

## Requirements

- Android 5.0 (Lollipop) or higher
- ~90MB storage
